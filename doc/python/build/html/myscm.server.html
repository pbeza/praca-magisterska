myscm.server.aidedbmanager module¶

class myscm.server.aidedbmanager.AIDEDatabasesManager(server_config)[source]¶

Bases: object

Manager of the AIDE aide.db[.X] databases directories (X is integer).

This manager is not AIDE’s database parser (see AIDECheckParser class). This class is responsible only for renaming AIDE’s databases after myscm-srv –scan to keep old databases saved (they are needed to run myscm-srv with –gen-img option).
