# mySCM-cli security README                                                   #
###############################################################################

This directory contains files related to the security of the mySCM system image
generated by the myscm-srv, specifically to the certificate that is used to
sign mySCM system images and RSA keys used for connecting to the client via
SFTP.

Files in this directory:

  * myscm-srv.cert.pem         - RSA private key for connecting to the client via SFTP,
  * myscm-srv.cert.pub.pem     - RSA public key for connecting to the client via SFTP.

See also:
